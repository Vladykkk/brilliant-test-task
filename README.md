# Brilliant Parallax

Front-end with awesome animations

## Technologies

- React v19
- SCSS
- Framer motion

## How to setup

1. Clone the repository:

```bash
git clone https://github.com/Vladykkk/brilliant-test-task.git
```

2. Install dependencies:

```bash
npm i
```

3. Run application

```bash
npm run dev
```
